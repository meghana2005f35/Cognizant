SELECT emp_name, department, CalculateAnnualSalary(monthly_salary) FROM employees WHERE CalculateAnnualSalary(monthly_salary) > 70000;

emp_name       | department | annual_salary
----------------|------------|---------------
Sanjay Reddy    | Sales      |   75000.00
Anaya Nair      | Sales      |   70000.00
Priya Singh     | IT         |   72000.00

EXPLANATION:
- Scalar function CalculateAnnualSalary multiplies monthly salary by 12
- Returns single value (DECIMAL type) for each input
- Can be used directly in SELECT clause
- Can be used in WHERE clause for filtering
- DETERMINISTIC keyword indicates function always returns same output for same input
- READS SQL DATA allows reading from tables
